组件渲染经历的生命周期

第一次渲染：

  • getDefaultProps
  • getInitialState
  • componentWillMount
  • render
  • componentDidMount

第二次渲染：

  • getInitialState
  • componentWillMount
  • render
  • componentDidMount

Props改变时：

  • componentWillReceiveProps
  • shouldComponentUpdate
  • componentWillUpdate
  • render
  • componentDidUpdate

State改变时：

  • shouldComponentUpdate
  • componentWillUpdate
  • render
  • componentDidUpdate

  • componentWillUnmont
